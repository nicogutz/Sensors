Hall effect magnetic sensors: Introduction
4
These sensors became quite cheap and have a lot of potential. They are still
not widespread usedbut they are rather new and booming. Moreoverthey
are very small.
How do hall effect magnetic sensors work?
A Hall sensor is a sensor which is sensitive to a magnetic field.
Is this the same as the inductive coil-based sensors?
Not really, in there we saw sensors for which the output voltage is sensitive
to the change of magnetic field (faradayâ€™s law). If there is no movement or
change in the magnetic field, the inductor will not see a voltage.
Hall sensors can detect a magnetic field, even if there is no movement.
Thus, they can detect a magnetic field even when it is not change. For
instance, they can be used to detect the gravitational field around the earth.
Of course, in order tohave this magnetic field, we will need a current going
through the sensor. The current will generate the magnetic field (Ampereâ€™s
law) and a voltage will be created.
Of course, this condition only happens in very specific materials. In fact, until
recently, this type of sensors could only be made out ofsome heavy metals
(such as gold and platinum). Now, there are actually semiconductorsthat
behave the same way.

Hall effect magnetic sensors: Application
7
Where can we use these hall sensors?
For instance, in industrial applications we can use them to detect the rotation
of a shaft contactlessly. On the shaft we place one or more magnets, and the
hall sensor close to the magnet. Each time the shaft turnsand the magnet
comes close to the hall sensor, a pulse will be detected.
How would we measure the rpms (rotation per minute)? We would put one
magnet and measure the period between pulses.
The hall sensor output would need to be amplified a little bit (as it is
millivolts), and a comparator with hysteresis is needed so that we can obtain
very nice pulses.
Is the hall sensor good for position detection?
Not really, it is very rough as we cannot really put infinite magnets on the
shaft, but it can actually detectthe displacement of the shaft.

Thus, depending on the type of movement, we will need to measure velocity
in a different manner.

small). We can measure the displacement and divide it by time. This will
be a mean velocity. Differentiate displacement to measure velocity.
â€¢ If the velocity is changing more during the measurement, the average will
be wrong. In that case, we can measure momentary velocity which is a bit
less obvious.

Acceleration sensors
We just said that for velocity we have an issue that we could not really
measure instantaneous velocity but rather average one by means of
displacement.
Acceleration sensors encounter also an issue.
It is not that easy to measure it because of the influence of the gravity force
(or acceleration). So, any accelerometer that we can build it will always
measure the gravity force or acceleration, where we actually wantto
measure is the relative acceleration of a device.
We will then need to differentiate between absolute acceleration and relative
acceleration sensors.
